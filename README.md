# Simple Simulation (Ren'Py + FSM)

Game ini dibuat dengan **Ren'Py** dan menggunakan pendekatan **Finite State Machine (FSM)** untuk mengatur alur cerita serta transisi antar keadaan game.

## Ringkas Konsep
- Setiap kondisi game direpresentasikan sebagai state.
- Perpindahan state terjadi berdasarkan pilihan pemain atau event tertentu.
- Struktur FSM membantu logika game lebih rapi dan mudah dikembangkan.

## Struktur Utama
- `game/script.rpy`: alur cerita dan logic utama.
- `game/screens.rpy`: tampilan UI/screen.
- `game/options.rpy`: konfigurasi proyek.

## Cara Menjalankan
1. Buka proyek ini melalui Ren'Py Launcher.
2. Pilih proyek `simplesimulation`.
3. Klik **Launch Project**.
